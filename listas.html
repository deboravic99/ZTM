A tag <ol> significa 'ordered list' e vai ler o conteúdo da seguinte forma, com a tag <li>, que significa 'list item':

  1. item 1
  2. item 2
  3. item 3

Para uma lista que não apresente os números ordenados, será preciso utilizar a tag <ul>, 'unordered list'.

  • item 1
  • item 2
  • item 3

Caso haja necessidade de colocar itens dentro de um item da lista, usa-se a tag da lista ordenada ou da lista desornada, porém como 'children tags', uma tag dentro de outra, com espaçamento. A tag que contém a 'children tag' se chama 'parenting tag'.

  • animais
    • gato
    • pato
    • abelha
  • item 2
  • item 3
